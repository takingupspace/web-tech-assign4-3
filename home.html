  • AWS MySQL RDS Instance used for data persistence
  • Insomnia used for API-testing, MySQL Workbench as an interfacing tool for the database

  • Customer searches for hotels by city: they can book a reservation, and delete their reservations
  • Hotel admins can add properties, add rooms, and create or delete reservations for customers
  • Chat-bot with Google's Dialogflow; Kommunicate used for taking over conversation with bot
  • Authentication implemented, ensuring only credentialed accounts can use certain features
